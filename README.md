# Superstore Sales Data Analysis

## Project Overview

This project analyzes Superstore sales data using Python, Pandas, and Matplotlib.

The goal of this project is to explore sales trends, customer segments, product performance, and shipping efficiency to generate business insights from e-commerce data.

---

## Tools & Libraries

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Dataset

Superstore Sales Dataset

Main fields used in this project:

- Order Date
- Ship Date
- Segment
- City
- Category
- Sub-Category
- Product Name
- Sales

---

## Project Workflow

### 1. Data Cleaning

- Loaded CSV data using Pandas
- Checked missing values
- Converted date columns to datetime format
- Created monthly features for trend analysis

### 2. Sales Trend Analysis

- Aggregated monthly sales
- Visualized sales trends over time
- Identified seasonal sales patterns

### 3. Regional Analysis

- Analyzed top-performing cities
- Compared sales across different regions

### 4. Customer Segment Analysis

- Compared sales by customer segment
- Calculated average order value by segment

### 5. Product Analysis

- Identified top-selling products
- Analyzed category and sub-category performance

### 6. Shipping Efficiency Analysis

- Calculated shipping duration
- Compared shipping speed across ship modes

---

## Key Insights

- Technology category generated the highest sales
- Consumer segment contributed the largest revenue
- Several cities accounted for a significant proportion of total sales
- Sales showed clear monthly trends and seasonality
- Shipping efficiency varied across shipping methods
